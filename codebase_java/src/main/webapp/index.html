Platform Overview.

IOT based Remote Mointor Controller system has ability to control the multiple electronic devices remotely. The system provides intelligent services for users and provides control with Smart Phone devices. The system consist of three types of devices.

Device List

Following Screen lists all the devices purchased by the user. Under this screen user has been provided ability to define device Formulas (Which is required to determine the state of device i.e, device state is high or low), Escalation (Which is a Rule by which users gets notified in case of system failures) and Device Control.

Device Details

This screen provides a user to configure Appliances over the respective analog inputs and digital outputs (for e.g, Pin A0 - Fan And D0 - Tv). Based on the device Type user able to add 16 or 8 appliances to Analog input and 8 or 4 appliances to Digital output.

Device Control

Following screen provides User to direct control over the devices. User has provided with three state of switch which is Auto: If this state is enabled then the control action over the appliances will be taken by the device itself in case of failures. If it is not enabled then action over the appliances will be take by the users directly by providing state ON or OFF.

Shutdown

Following screen provides User to decide the shutdown action over the device. We provide ability for the users to decide the action, if device goes in bad state. Providing with two options either not to shutdown else shutdown and let operator decide to turn it ON.

Escalation

Device Escalation provides email notification to the users based on ceratian rule provided by user, if selected device met under the Rules then email is trigrred to the selected group to taken action.

Email Notification

Email functionality is provided with Group Name, email and Message. Corresponding group will take action on device in case of system failures.

On Boarding Process

  • #1 Registration of Device in Factory

    Device Initialization and Registration. Based on device type either 8 or 16 analog inputs are connected to the platform and 4 to 8 devices can be controlled.

  • #2 Installation of Device

    Dispatch our service engineers to your place to set controls of the device based on your requirement.

  • #3 Customization of Device

    The user has a control to configure the Formula for Device Output based on Analog inputs, by which state of the device is determined which is either ON/OFF and Escalation email can be configured and triggered to Group in case of failures.

  • #4 Monitoring State of Device

    The continues Analog reading of each device has been monitored for every X interval of minutes, which will determine the state of the device.
